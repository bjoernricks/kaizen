jam by bjoernricks

An advanced package manager for Free Software

Jam is a package manager for Free Software targeting mostly Mac OS X. Of course there are several good package managers for Mac OS like MacPorts, Homebrew and others but they don't fit my personal needs and are written in programming languages outside of my scope. Also it's a lot of fun to write such a software from scratch and learning a lot about python. So try out jam and start to orchestrate your Free Software!
