kaizen by Björn Ricks

Continously improve, build and manage free software

Kaizen is a manager for free software targeting mostly Mac OS X. Of course there are several good package managers for Mac OS like MacPorts, Homebrew and others but they don't fit my personal needs and are written in programming languages outside of my scope. Also it's a lot of fun to write such a software from scratch and learning a lot about python.

So try out kaizen and start to orchestrate your free software!
